# Twelve Ninja Art Asset Prompts

This file contains Gemini prompts for generating game assets for Twelve Ninja. The style is a "Dark Sumi-e" ink wash aesthetic, characterized by high contrast, minimalist color palettes, and traditional Japanese brush painting techniques.

---

### 1. Player Avatar: "Silent Shadow Ninja"

**Description:** A full-body character sprite for the main player. The ninja should appear agile, mysterious, and highly skilled.

**Gemini Prompt:**
```
Full-body character sprite for a browser-based MMORPG, silent shadow ninja, agile and mysterious pose, web-optimized, transparent background, PNG.

**Art Style:**
Dark Sumi-e ink wash, Japanese brush painting, high-contrast silhouette, minimalist black and grey palette with a single deep crimson accent on the scarf, dynamic ink splatters suggesting movement, character defined by strong, fluid brushstrokes.

**Composition:**
Centered, full-body shot, character is in a low-crouch, ready to spring into action, holding a short tanto blade. The focus is on the silhouette and the flow of the character's clothing.

**Negative Prompt:**
photorealistic, 3D render, complex colors, vibrant, detailed background, heavy armor, bulky.
```

### 3. World Tile: "Cursed Swamp"

**Description:** A hexagonal background tile for a dangerous, mystical swamp area on the world map.

**Gemini Prompt:**
```
Hexagonal world map tile for a browser-based MMORPG, top-down perspective, cursed swamp terrain, seamless tiling with adjacent tiles, web-optimized, 512x512 resolution.

**Art Style:**
Dark Sumi-e ink wash, Japanese brush painting, minimalist landscape, gnarled, dead trees rendered as black silhouettes, pools of stagnant water suggested with gray washes and subtle ripples, wisps of fog created with diluted ink effects.

**Composition:**
Top-down view, pointy-top hexagon orientation. The visual weight is balanced, with a few key elements (a twisted tree, a dark pool) providing interest without cluttering the tile. Edges should fade slightly to allow for natural blending.

**Negative Prompt:**
photorealistic, 3D render, bright colors, lush greenery, living trees, clear water, detailed ground textures.
```

### 4. UI Icon: "Health Potion"

**Description:** A simple, clear icon for a health potion item, to be used in the inventory and HUD.

**Gemini Prompt:**
```
UI icon for an RPG inventory, health potion in a small gourd flask, SVG format, web-optimized, transparent background.

**Art Style:**
Dark Sumi-e ink wash, single, bold brushstroke defining the gourd's silhouette, a simple red circle on the front as a minimalist label, high-contrast black and white with one color accent, easily readable at a small scale (32x32).

**Composition:**
Icon is centered, simple, and instantly recognizable. The shape is clean and symbolic rather than detailed.

**Negative Prompt:**
photorealistic, 3D render, glass bottle, complex reflections, multiple colors, text, detailed shading.
```

### 5. UI Panel: "Parchment Menu Background"

**Description:** A flexible background panel for UI menus like inventory, character stats, or quest logs.

**Gemini Prompt:**
```
UI menu background, 9-slice scaleable asset, browser-based RPG, web-optimized, PNG.

**Art Style:**
Dark Sumi-e, looks like old, worn parchment or rice paper, subtle texture, dark ink-stained borders, minimalist style, a faint bamboo stalk silhouette in one corner, predominantly light tan/beige with dark gray/black borders. The edges should look like brushed ink.

**Composition:**
A rectangular panel with thicker, darker ink wash borders that can be sliced for scaling. The center is mostly clear to allow text to be readable on top.

**Negative Prompt:**
photorealistic, 3D render, clean edges, modern, digital-looking, vibrant colors, heavy patterns.
```

### 8. Map Art: "Mountain Monastery Landmark"

**Description:** An illustrative art piece for a major landmark on the world map.

**Gemini Prompt:**
```
World map landmark illustration, Japanese monastery perched on a snowy mountain cliff, browser-based RPG, web-optimized, PNG with transparent background.

**Art Style:**
Dark Sumi-e ink wash landscape painting, minimalist and atmospheric, strong vertical brushstrokes for the mountain cliffs, pagoda roofs defined by simple, dark silhouettes, snow suggested by negative space, swirling ink wash for clouds around the peak.

**Composition:**
The monastery is the focal point, but it is small compared to the vastness of the mountain. The composition should create a sense of scale, isolation, and tranquility.

**Negative Prompt:**
photorealistic, 3D render, vibrant colors, lush greenery, close-up shot, modern architecture.
```
